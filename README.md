## MyOS介绍

**运行环境**

*   Ubuntu 16.04
*   Bochs 2.6



**映像制作**

```
工作目录下键入
make image
再将bochs配置文件bochsrc.bxrc导入到bochs
将生成的kernel.img放入bochs运行
```
